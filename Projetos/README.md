# Nivel Iniciante
    1. Calculadora no Terminal -> operações básicas (+,-,/,*) , Menu interativo
    2. Gerador de senhas aleatorias -> escolher tamanho , incluindo letras e números e simbolos,mostrar senhas fortes
    3. Lista de tarefas -> adicionar tarefas , listar e remover tarefas , salvar em arquivo txt. ou json.
    4. Jogo de adivinhação -> computador escolhe um número aleatorio , contador de tentativas
    5. Conversor de unidades -> temperatura , moeda e distancia

# Nivel Intermediario
    1. Sistemas de cadastro de usuarios -> Criar ,editar e excluir usuarios,login e senha , salvar json ou sqlite
    2. Web scraper -> Extrair titulos de sites , buscar preços de produtos,salvar dados em planilhas(dica : pandas)
    3. Bot de automações -> organizar arquivos automaticamente,renomear arquivos em lote , automatizar tarefas repetivas do pc
    4. Jogo com interface gráfica -> jogo da velha,pedra,papel,tesoura,usar Tkinter ou PyGame
    5. Gerador de relatórios -> Ler dados de um arquivo CSV , Calculadora estatísticas , gerar um relatório em Pdf

# Nivel Avançado
    1. API REST completa -> Criar usuarios , autenticação JWT , CRUD Completo , usar Flask ou FastAPI
    2. Sustema web completo -> Login e cadastro , painel administrativo , banco de dados , deploy online
    3. Chat em tempo real -> comunicação entre usuarios , websockets, interface web
    4. Inteligencia artificial simples -> reconhecimento de imagens , previsao de dados , chatbot treinado
    5. Sistema de recomendação -> Recomendar produtos , baseado em preferencias do usuario , similar ao que lojgas online usam

    1. Criar um mini sistema operacional em terminal
    2. Criar um interpretador de linguagem
    3. Criar um jogo 2D completo estilo RPG
